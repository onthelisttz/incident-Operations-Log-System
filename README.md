# Incident & Operations Log System (IOLS)

## Overview

The **Incident & Operations Log System (IOLS)** is a comprehensive web-based platform designed to reporting, tracking, and resolving operational incidents. It facilitates efficient incident management for organizations through role-based access control and detailed analytics.

The system is built using a modern decoupled architecture:
- **Backend**: Laravel (PHP) - providing a robust RESTful API.
- **Frontend**: React (TypeScript) - delivering a responsive and dynamic user interface.

## User Roles

- **Admin**: Full system access, including user management.
- **Operator**: Manages and resolves assigned incidents.
- **Reporter**: Reports incidents and tracks their status.

Before setting up the project, ensure you have the following installed:

- **Node.js** (v18 or higher) & **npm**
- **PHP** (v8.2 or higher)
- **Composer**
- **MySQL** (v8.0 or higher)
